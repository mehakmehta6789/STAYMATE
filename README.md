# STAYMATE 🏠🎓
**StayMate** is a web platform that helps students find suitable **PG accommodations and compatible roommates near their campus**.  
The platform connects students with PG owners, making it easier to discover housing, view facilities, and send stay requests.

---

## 📌 Project Overview

Finding safe and affordable accommodation near universities can be difficult for students.  
**StayMate** simplifies this process by providing a centralized platform where students can explore PG listings, view rent and facilities, and request to stay.

The platform supports **two main user roles**:

### 👨‍🎓 Students
- Browse PG listings
- View PG details and facilities
- Send requests to join PGs
- Discover potential roommates

### 🏠 PG Owners
- Add and manage PG listings
- View student requests
- Approve or reject requests
- Manage room availability

---

## ✨ Features

- Role-based authentication (**Student & PG Owner**)
- PG listings with **images, rent, facilities, and location**
- Student request system to join PGs
- Roommate discovery
- Owner dashboard to manage PG properties
- Availability tracking and approved residents
- Google Maps integration for PG location
- Responsive and clean user interface

### 🧭 Installation Steps

#### 1. Clone the Repository
```bash
git clone https://github.com/mehakmehta6789/STAYMATE.git

```
#### 2. Navigate into the project folder
```bash
cd staymate

```
#### 3. Install Dependencies
```bash
npm install

```

#### 4. Create a .env file
```bash
PORT=3000
MONGO_URI=your_mongodb_connection
SESSION_SECRET=your_secret_key

```

#### 5. Run the Application
```bash
npm start
```
